# supermarket_analysis
Supermarket sales analysis focused on city-wise, category-wise, and customer-type performance to identify operational gaps and provide actionable, non-promotional business insights using Python
Customer Purchase Behavior Analysis
Project Overview

This project analyzes customer purchase behavior across cities, product categories, and customer types (Member vs Normal) to identify revenue patterns, conversion gaps, and opportunities for targeted business interventions.

The analysis focuses on understanding:

Category-level performance by city

Differences in spending behavior between Members and Non-Members

Areas with untapped membership conversion potential

Opportunities to improve basket size and category visibility
Dataset Summary

The dataset contains transactional-level retail data with attributes including:

City

Product Category

Customer Type (Member / Normal)

Transaction Amount

Aggregations and pivots are used to evaluate revenue concentration and customer segmentation trends.

Analytical Approach

The analysis was conducted using:
GroupBy and Pivot Tables for segmentation

Membership vs non-membership comparisons

Category-wise revenue contribution by city

Identification of missing or low-penetration segments

Pandas was used for data manipulation and summarization.
